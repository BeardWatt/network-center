## 功能

自动根据照片的EXIF信息，在照片文件名后添加拍摄日期（月、日）和时间（时、分）

## 运行方式

```bash
python3 main.py
```

## 注意事项

手机拍摄的照片可以使用微信，发送原图到电脑。

iPhone 7之后的机型，默认照片格式为HEIC，通过微信发送到电脑会丢失EXIF信息，解决方法待寻
